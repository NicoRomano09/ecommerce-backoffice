# Planificación

Esta carpeta contiene la documentación relacionada con la planificación y organización del proyecto.

## Contenido

Aquí se almacenarán los documentos utilizados durante el desarrollo, tales como:

* Historias de usuario.
* Casos de uso.
* Planificación de funcionalidades.
* Registro de decisiones de diseño.
* Otra documentación funcional del proyecto.

## Objetivo

Centralizar la documentación utilizada para definir, organizar y dar seguimiento al desarrollo del sistema, permitiendo mantener un registro de las decisiones tomadas durante el proyecto.
