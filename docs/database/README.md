# Base de Datos

Esta carpeta reúne la documentación relacionada con la base de datos del proyecto.

## Contenido

* **Modelo Relacional**: diagrama de la estructura de la base de datos, incluyendo entidades, atributos y relaciones.
* **Script SQL**: archivo con la definición de la base de datos y la creación de las tablas.

## Objetivo

La documentación almacenada en esta carpeta sirve como referencia para comprender el diseño de la base de datos y facilitar su implementación y mantenimiento durante el desarrollo del sistema.
