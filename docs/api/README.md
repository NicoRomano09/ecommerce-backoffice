# API

Esta carpeta contiene la documentación de la API del sistema.

## Contenido

Aquí se documentarán los distintos endpoints desarrollados durante el proyecto, incluyendo:

* Métodos HTTP utilizados.
* Rutas disponibles.
* Parámetros de entrada.
* Estructura de las solicitudes y respuestas.
* Códigos de estado HTTP.
* Ejemplos de uso.

## Objetivo

Proporcionar una referencia técnica para facilitar el desarrollo, el mantenimiento y la integración entre el frontend y el backend.
