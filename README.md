# Ecommerce Backoffice

## Descripción

**Ecommerce Backoffice** es un proyecto Full Stack desarrollado como parte de mi portfolio profesional.

El objetivo es diseñar e implementar un sistema de Backoffice para operaciones de ecommerce, permitiendo gestionar pedidos, incidencias e indicadores del negocio mediante una aplicación web orientada al uso interno de una organización.

## Alcance del proyecto

La primera versión del sistema contempla los siguientes módulos:

* Dashboard Operativo.
* Gestión de Pedidos.
* Centro de Incidencias.
* Indicadores del Negocio.

## Estructura del repositorio

```text
backend/   → Desarrollo de la API y lógica de negocio.
frontend/  → Desarrollo de la interfaz de usuario.
docs/      → Documentación técnica y funcional del proyecto.
```
